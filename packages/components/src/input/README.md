# Input

A customizable input field component with variants and sizes.

## Installation

```bash
npx @dinachi/cli@latest add input
```

## Usage

```tsx
import { Input } from "@/components/ui/input"
```

```tsx
<Input type="email" placeholder="you@example.com" />
```

## API Reference

- **Input** -- Styled wrapper around `Input` from Base UI. Renders an `<input>` element with focus ring, placeholder styling, file input support, and validation states. Accepts all standard HTML input attributes.
