# Textarea

A multi-line text input for longer form content.

## Installation

```bash
npx @dinachi/cli@latest add textarea
```

## Usage

```tsx
import { Textarea } from "@/components/ui/textarea"
```

```tsx
<Textarea placeholder="Type your message here." />
```

## API Reference

- **Textarea** -- A styled `<textarea>` element. Accepts all standard HTML textarea attributes including `placeholder`, `rows`, `disabled`, and `value`/`onChange` for controlled usage.
