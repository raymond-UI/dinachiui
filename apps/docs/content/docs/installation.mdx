## Quick Start (Any Framework)

If you are using a different framework or want to get started quickly, follow these three steps:

<Steps>

<Step title="Initialize DinachiUI">

Run the init command to set up your project configuration and install base dependencies:

```bash
npx @dinachi/cli@latest init
```

</Step>

<Step title="Add Components">

Install the components you need. You can add one or multiple at a time:

```bash
npx @dinachi/cli@latest add button
```

</Step>

<Step title="Import and Use">

Import the installed component and start using it in your project:

```tsx
import { Button } from "@/components/ui/button";
```

</Step>

</Steps>
