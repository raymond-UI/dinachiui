## Installation

<Steps>

<Step title="Create Vite Project">

Start with a new Vite project or use an existing one.

**Create a new React + TypeScript project:**

```bash
npm create vite@latest my-app -- --template react-ts
```

**Navigate to your project and install dependencies:**

```bash
cd my-app && npm install
```

<Callout tone="info">
You can also run `npm create vite@latest` without arguments for an interactive prompt to choose your framework and variant.
</Callout>

</Step>

<Step title="Install Tailwind CSS">

Set up Tailwind CSS if it is not already configured in your project.

**Install Tailwind CSS and the Vite plugin:**

```bash
npm install tailwindcss @tailwindcss/vite
```

**Add the plugin to `vite.config.ts`:**

```ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
  ],
})
```

**Replace the contents of `src/index.css` with:**

```css
@import "tailwindcss";
```

<Callout tone="info" title="Using Tailwind CSS v3?">
If your project uses Tailwind CSS v3, install with `npm install -D tailwindcss postcss autoprefixer` and run `npx tailwindcss init -p` instead. The `init` command will detect your Tailwind version and configure accordingly.
</Callout>

</Step>

<Step title="Configure Path Aliases">

Set up the `@/` path alias so you can use clean imports like `@/components/ui/button`.

**Update `vite.config.ts`:**

```ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'
import path from 'path'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
  ],
  resolve: {
    alias: {
      "@": path.resolve(__dirname, "./src"),
    },
  },
})
```

<Callout tone="info">
The `init` command in the next step will automatically add the `@/*` path mapping to your `tsconfig.app.json`. You only need to set up the Vite resolve alias here.
</Callout>

</Step>

<Step title="Initialize DinachiUI">

Run the init command to set up DinachiUI configuration and install dependencies:

```bash
npx @dinachi/cli@latest init
```

<Callout tone="info" title="What this does">
The init command automatically:

- Creates a `components.json` config file with your project settings
- Injects the full color theme (light + dark mode CSS variables) into your CSS file
- Sets up the `@/*` path alias in your tsconfig
- Creates the `cn()` utility function
- Installs required dependencies (clsx, tailwind-merge, class-variance-authority)
</Callout>

</Step>

<Step title="Add Components">

Install the components you need for your project. You can add one or multiple at a time:

```bash
npx @dinachi/cli@latest add button input card
```

Or install all components at once:

```bash
npx @dinachi/cli@latest add --all
```

</Step>

<Step title="Start Using Components">

Import and use DinachiUI components in your Vite app.

**Update `src/App.tsx`:**

```tsx
import { useState } from 'react'
import { Button } from '@/components/ui/button'
import { Card, CardContent, CardHeader, CardTitle } from '@/components/ui/card'

function App() {
  const [count, setCount] = useState(0)

  return (
    <div className="container mx-auto p-8">
      <Card className="max-w-md mx-auto">
        <CardHeader>
          <CardTitle>DinachiUI + Vite</CardTitle>
        </CardHeader>
        <CardContent className="space-y-4">
          <Button onClick={() => setCount(count + 1)}>
            Count: {count}
          </Button>
        </CardContent>
      </Card>
    </div>
  )
}

export default App
```

**Start the development server:**

```bash
npm run dev
```

</Step>

</Steps>

<Callout tone="success" title="You're all set!">
Your Vite project is now configured with DinachiUI. Visit [the components page](/docs/components) to explore all available components and start building your UI.
</Callout>
