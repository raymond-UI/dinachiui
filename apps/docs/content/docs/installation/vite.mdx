<Steps>

<Step title="Create Vite Project">

<PackageManagerTabs
  npm="npm create vite@latest my-app -- --template react-ts"
  pnpm="pnpm create vite@latest my-app --template react-ts"
  yarn="yarn create vite my-app --template react-ts"
  bun="bunx create-vite my-app --template react-ts"
/>

<PackageManagerTabs
  npm="cd my-app && npm install"
  pnpm="cd my-app && pnpm install"
  yarn="cd my-app && yarn install"
  bun="cd my-app && bun install"
/>

</Step>

<Step title="Install Tailwind CSS">

<PackageManagerTabs type="install" command="tailwindcss @tailwindcss/vite" />

**Add the plugin to `vite.config.ts`:**

```ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
  ],
})
```

**Replace the contents of `src/index.css` with:**

```css
@import "tailwindcss";
```

<Callout tone="info" title="Using Tailwind CSS v3?">
Install with `npm install -D tailwindcss postcss autoprefixer` and run `npx tailwindcss init -p` instead. The CLI detects your Tailwind version automatically.
</Callout>

</Step>

<Step title="Configure Path Aliases">

**Update `vite.config.ts`:**

```ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'
import path from 'path'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
  ],
  resolve: {
    alias: {
      "@": path.resolve(__dirname, "./src"),
    },
  },
})
```

The `init` command in the next step adds the `@/*` mapping to your `tsconfig.app.json` automatically.

</Step>

<Step title="Initialize DinachiUI">

<PackageManagerTabs type="exec" command="@dinachi/cli@latest init" />

This creates `components.json`, injects the color theme into your CSS, and installs base dependencies.

</Step>

<Step title="Add Components">

<PackageManagerTabs type="exec" command="@dinachi/cli@latest add button input card" />

Or install everything:

<PackageManagerTabs type="exec" command="@dinachi/cli@latest add --all" />

</Step>

<Step title="Start Building">

```tsx
import { useState } from 'react'
import { Button } from '@/components/ui/button'

function App() {
  const [count, setCount] = useState(0)

  return (
    <Button onClick={() => setCount(count + 1)}>
      Count: {count}
    </Button>
  )
}

export default App
```

<PackageManagerTabs type="run" command="dev" />

</Step>

</Steps>
