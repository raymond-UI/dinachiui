<Callout tone="info" title="How it works">
Unlike traditional component libraries, the DinachiUI CLI copies actual source code into your project. You get full ownership — modify variants, styles, and behavior freely.
</Callout>

## Commands

### `init`

Set up DinachiUI in your project. Creates `components.json`, injects the color theme, installs base dependencies.

<PackageManagerTabs type="exec" command="@dinachi/cli@latest init" />

### `add`

Add components to your project. Copies source code and installs dependencies automatically.

<PackageManagerTabs type="exec" command="@dinachi/cli@latest add button" />

Add multiple at once:

<PackageManagerTabs type="exec" command="@dinachi/cli@latest add button input card" />

Install all components:

<PackageManagerTabs type="exec" command="@dinachi/cli@latest add --all" />

Overwrite existing files:

<PackageManagerTabs type="exec" command="@dinachi/cli@latest add button --overwrite" />

### Global Install

For shorter commands, install globally:

<PackageManagerTabs type="global" command="@dinachi/cli" />

Then use `dinachi init` and `dinachi add button` directly.

## Configuration

After running `init`, your project will have a `components.json`:

```json
{
  "style": "default",
  "rsc": true,
  "tsx": true,
  "tailwind": {
    "css": "src/app/globals.css",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui",
    "lib": "@/lib"
  }
}
```
