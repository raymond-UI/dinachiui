<Callout tone="info" title="How it works">
Unlike traditional component libraries, DinachiUI CLI copies the actual source code into your project. This gives you full ownership and control over your components.

- **Full ownership** -- Code is copied to your project
- **Installs only what you need** -- Component dependencies are added automatically
- **Complete customization** -- Modify variants, styles, and behavior
- **TypeScript support** -- Full type safety out of the box
</Callout>

## Commands

### `init`

Initialize DinachiUI in your project. Creates configuration file and installs dependencies.

```bash
npx @dinachi/cli@latest init
```

### `add [component]`

Add a specific component to your project. Copies source code and installs dependencies.

```bash
npx @dinachi/cli@latest add button
```

### `add --all`

Install all available components at once.

```bash
npx @dinachi/cli@latest add --all
```

### `add --overwrite`

Overwrite existing component files.

```bash
npx @dinachi/cli@latest add button --overwrite
```

## Configuration

After running `init`, you will have a `components.json` file:

```json
{
  "style": "default",
  "rsc": true,
  "tsx": true,
  "tailwind": {
    "config": "tailwind.config.ts",
    "css": "src/app/globals.css",
    "baseColor": "slate",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui",
    "lib": "@/lib",
    "hooks": "@/hooks"
  },
  "iconLibrary": "lucide"
}
```

## Examples

### Setup new project

```bash
# Initialize DinachiUI
npx @dinachi/cli@latest init

# Add components you need
npx @dinachi/cli@latest add button input card

# Start using them
import { Button } from '@/components/ui/button'
```

### Add all components

```bash
# Install all components at once
npx @dinachi/cli@latest add --all

# Or install globally for shorter commands
npm install -g @dinachi/cli
dinachi add --all
```

### Update existing component

```bash
# Overwrite existing component with latest version
npx @dinachi/cli@latest add button --overwrite

# This will replace your existing button component
# Make sure to backup any customizations first!
```
